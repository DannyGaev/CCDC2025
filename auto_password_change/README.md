```powershell
   oaktree@OakTree:~/CCDC2025/auto_password_change$ ./change_user_passwds.sh
```

```powershell

   [sudo] password for oaktree:

   nobody     gm*kdajj0k!v709m
   oaktree    !e&mh5k^r!2ns*ze
```
